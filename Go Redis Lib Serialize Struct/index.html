Summary

go-redis 库对于 HSet 方法支持的数据类型为 interface{}，在传递数据时会调用 writeCmd 方法进行序列化处理，最终通过 conn 对象将数据传递给 redis 服务端

支持的数据类型基本为基础数据类型，如果是自定义的结构体需要自己实现序列化和反序列化的方式
